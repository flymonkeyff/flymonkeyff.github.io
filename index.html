한글 멜로디 생성기

한글을 입력하고 마우스를 움직여 음악과 비주얼을 만들어보세요.
TouchDesigner와 실시간으로 연동됩니다.

사용 방법:

  • TouchDesigner를 먼저 실행하고 WebSocket 서버를 설정해주세요
  • 시작하기 버튼을 눌러 오디오를 활성화하세요
  • 한글을 입력하면 소리와 함께 TouchDesigner로 신호가 전송됩니다
  • 마우스를 움직여 음의 높이와 세기를 조절할 수 있습니다

TouchDesigner 설정:

WebSocket DAT → Mode: Server → Port: 9980
